Introduction

Base URL: https://api.raffled.live 
This documentation aims to provide all the information you need to work with our API.

As you scroll, you will see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile).
You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

To authenticate requests, include a Authorization header with the value "Bearer {YOUR_AUTH_TOKEN}".

All authenticated endpoints are marked with a requires authentication badge in the documentation below.

You can obtain your token by completing the SIWE process using the /siwe/nonce and /siwe/verify endpoints.

Stream on-chain events (SSE).

Opens a persistent Server-Sent Events connection that pushes every raffle event in blockchain order (block_number ASC → log_index ASC).

On first connection (no cursor) the server replays the latest 50 events and then polls every 3 seconds for new ones. Use the Last-Event-ID header (format: {block_number}:{log_index}) or the last_block / last_log_index query params to resume from a cursor.

Each pushed message has the SSE event name event_log:

id: 39000100:3
event: event_log
data: {"id":42,"event_type":"TicketPurchased",...}

A ": heartbeat" comment is sent when there are no new events to keep the connection alive.

Example request:
curl --request GET \
    --get "https://api.raffled.live/api/events/stream?last_block=39000100&last_log_index=3&event_type=TicketPurchased" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "https://api.raffled.live/api/events/stream"
);

const params = {
    "last_block": "39000100",
    "last_log_index": "3",
    "event_type": "TicketPurchased",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());
